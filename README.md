# 💰 Future Wallet – Personal Finance Tracker

## 📱 Overview

**Future Wallet** is a Kotlin-based personal finance tracking application developed using **Android Studio**. The app helps users manage their daily finances by tracking income and expenses, setting monthly budgets, and monitoring spending habits. All data is stored securely on the device, ensuring privacy and offline accessibility.

---

## ✨ Features

### 💸 Income & Expense Management

* Add and track income transactions
* Record daily expenses
* View transaction history

### 📊 Spending Analysis

* Categorize expenses for better financial insights
* Analyze spending patterns by category
* Monitor financial habits over time

### 📅 Budget Planning

* Set monthly budget limits
* Track budget utilization
* Receive alerts when approaching budget limits

### 🔒 Secure Local Storage

* Store data securely using SharedPreferences
* Offline access without requiring an internet connection

### 💾 Backup & Restore

* Create backups of financial records
* Restore saved data from internal storage

### 🔔 Notifications

* Receive spending reminders
* Stay on track with financial goals through timely alerts
